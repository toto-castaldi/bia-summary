1. Create `src/cli.ts` with the following:

   **Imports:**
   - `Command` from `"commander"` (Commander 14)
   - `validateEnv` from `"./config.js"`
   - `runPipeline` from `"./pipeline.js"`
   - `access` from `"node:fs/promises"` (to check PDF file exists)
   - `path` from `"node:path"`

   **Commander setup:**
   ```typescript
   const program = new Command();

   program
     .name("bia-summary")
     .description("Generate a structured Italian summary from a BIA report PDF")
     .version("0.1.0")
     .argument("<pdf-file>", "Path to the BIA report PDF")
     .option("--dry-run", "Preview markdown output without PDF conversion", false)
     .action(async (pdfFile: string, options: { dryRun: boolean }) => {
       // ... action body below
     });

   program.parse();
   ```

   **Action body (inside `.action()`):**

   Step 1 -- Validate environment (D-07, D-08):
   - Call `validateEnv()` -- this exits with code 1 if keys are missing

   Step 2 -- Validate PDF file exists:
   - Resolve path: `const inputPath = path.resolve(pdfFile)`
   - Try `await access(inputPath)`. On failure:
     - `console.error(\`Error: File not found: ${inputPath}\`)`
     - `process.exit(1)`

   Step 3 -- Run pipeline:
   - Call `const result = await runPipeline({ inputPath, dryRun: options.dryRun })`

   Step 4 -- Output result (D-04, D-05):
   - Print metadata header to stderr: `console.error(\`\nModel: ${result.model} | Tokens: ${result.inputTokens} in / ${result.outputTokens} out | Stop: ${result.stopReason}\`)`
   - Print markdown to stdout: `process.stdout.write(result.markdown)`

   Note: In Phase 1, the `--dry-run` flag exists but all paths lead to stdout output (no PDF conversion yet). The flag is forward-compatible for Phase 2.

   **Top-level error handling:**
   Wrap the entire action in try/catch. On uncaught error:
   - `console.error(\`Error: ${error instanceof Error ? error.message : String(error)}\`)`
   - `process.exit(1)`

   This ensures CLI-06: exit code 0 on success (Commander default), non-zero on any failure.

2. Run `npx tsc --noEmit` to verify compilation.
